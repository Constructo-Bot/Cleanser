## HOW TO USE THIS GUIDE

1. **Find the text you see on the website**
2. **Look up the corresponding file and line number in this guide**
3. **Open that HTML file and go to the specified line**
4. **Edit the text directly in the HTML code**

### Quick Search Tips:
- Use Ctrl+F (or Cmd+F on Mac) to search for specific text in the HTML files
- Look for the text within HTML tags like `<h1>`, `<p>`, `<span>`, etc.
- Remember that some text might be split across multiple lines in the code
- Check both the opening and closing tags when looking for text content
